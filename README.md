## 🚀 软件特征

### 核心功能
* ✅ **文本消息** - 支持最多144字符的消息发送，实时显示剩余可用字符数
* ✅ **历史记录** - 保存最近20条发送记录
* ✅ **自动发送** - 可设置5-300秒的间隔自动发送
* ✅ **OSC端口配置** - 支持自定义OSC通信端口

### 智能附加项
* ✅ **当前电脑时间** - 自动附加 UTC+8 格式时间
* ✅ **当前窗口标题** - 获取当前焦点窗口名称（支持字符数限制）
* ✅ **音乐信息** - 使用 SMTC 协议获取歌曲信息（支持网易云音乐等，可限制标题/艺术家字符数）
* ✅ **挂机检测** - 检测离开电脑的时长，5-21600秒阈值可调，超过阈值自动显示
* ✅ **硬件监测** - CPU、RAM、GPU 占用信息，支持自定义标签不支持AMD显卡）
* ✅ **心率监测** - 蓝牙直连手环/手表获取实时心率

### 个性化设置
* ✅ **发送顺序调整** - 自由排列消息内容与各项附加信息的先后顺序
* ✅ **硬件标签自定义** - 自定义硬件监测项的显示名称
* ✅ **字符数限制** -控制窗口标题和音乐信息的显示长度
* ✅ **发送设置** - 控制OSC发送的地址以及端口
* ✅ **模板字符串模式** - 高级消息格式化功能，支持多种变量（**NEW!**）
* ✅ **高级音乐信息** - 网易云音乐歌词同步（**NEW!**）

### ❤️ 心率监测使用说明

* 支持的设备
* 小米手环 8/9/10
* 小米手表
* 其他支持 HRS (Heart Rate Service) 协议的蓝牙心率设备

### 使用方法
1. 确保你的设备已开启心率广播功能
2. 在程序中勾选"心率检测"选项
3. 程序会自动扫描并连接附近的蓝牙心率设备
4. 连接成功后，心率数据会显示在调试面板中
5. 发送消息时会自动附加 [❤️:XXX BPM] 格式的心率信息

### ⚙️ 设置功能说明（v1.3新增）

点击主界面"设置"按钮可打开设置窗口，包含四个标签页：

   ### 发送顺序
  * 通过上移/下移按钮调整各项内容的排列顺序
  * 支持排序的项包括：消息内容、时间、挂机状态、窗口标题、音乐信息、心率、硬件监测
  * 在模板模式下，此功能会被禁用
  ### 硬件标签
  * 自定义CPU、RAM、GPU的显示标签
  * 留空则使用默认格式（如"CPU: 45%"）
  * 设置标签后格式变为"CPU(标签): 45%"
  ### 字符限制
  * 窗口标题最大字符数：1-100（默认20）
  * 音乐标题最大字符数：1-100（默认30）
  * 音乐艺术家最大字符数：1-100（默认30）

  ### 进阶音乐信息（v1.3新增）
  * 启用高级音乐信息（替换普通音乐信息）
  * 提供启动网易云、选择网易云路径、开始同步歌词等功能
  * 可自定义进度条宽度、填充字符、滑块字符、空白字符
  * 支持自定义音乐信息模板，使用{song}, {artist}, {bar}, {time}, {lyric1}, {lyric2}变量

  ### (高级)发送顺序（v1.3新增）
  * 启用模板字符串模式，使用高级格式化功能
  * 提供快速插入附加项的按钮，包括消息内容、时间、窗口标题、挂机状态、音乐信息、硬件监测、心率、换行符等
  * 可自定义模板字符串，支持UTF-8字符
