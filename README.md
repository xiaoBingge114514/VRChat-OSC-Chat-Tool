## ✨ 最新更新（v1.2 - 2025.3.5）

### 🎉 新增功能
- **OSC端口自定义** - 支持修改OSC通信端口（默认9000），适应不同VRChat配置需求
- **发送顺序自定义** - 可自由调整消息内容与附加项（时间/窗口/音乐/心率/硬件等）的发送顺序
- **硬件标签自定义** - 支持自定义CPU、RAM、GPU的显示标签（如改为"处理器"、"内存"等）
- **字符数限制设置** - 可分别设置窗口标题、音乐标题、音乐艺术家的最大显示字符数
- **配置保存** - 使用文件保存启用的功能、发送顺序、硬件标签

### 🔧 优化改进
- 挂机检测阈值范围扩大为5-21600秒（原60-21600秒），支持更短的检测时间
- 默认发送间隔调整为10秒（原15秒）
- 优化了硬件监测的显示格式

## 🚀 软件特征

### 核心功能
* ✅ **文本消息** - 支持最多144字符的消息发送，实时显示剩余可用字符数
* ✅ **历史记录** - 保存最近20条发送记录
* ✅ **自动发送** - 可设置5-300秒的间隔自动发送
* ✅ **OSC端口配置** - 支持自定义OSC通信端口（**NEW!**）

### 智能附加项
* ✅ **当前电脑时间** - 自动附加 UTC+8 格式时间
* ✅ **当前窗口标题** - 获取当前焦点窗口名称（支持字符数限制）
* ✅ **音乐信息** - 使用 SMTC 协议获取歌曲信息（支持网易云音乐等，可限制标题/艺术家字符数）
* ✅ **挂机检测** - 检测离开电脑的时长，5-21600秒阈值可调，超过阈值自动显示
* ✅ **硬件监测** - CPU、RAM、GPU 占用信息，支持自定义标签（*不支持AMD显卡*）
* ✅ **心率监测** - 蓝牙直连手环/手表获取实时心率

### 个性化设置
* ✅ **发送顺序调整** - 自由排列消息内容与各项附加信息的先后顺序（**NEW!**）
* ✅ **硬件标签自定义** - 自定义硬件监测项的显示名称（**NEW!**）
* ✅ **字符数限制** - 控制窗口标题和音乐信息的显示长度（**NEW!**）

### ❤️ 心率监测使用说明

* 支持的设备
* 小米手环 8/9/10
* 小米手表
* 其他支持 HRS (Heart Rate Service) 协议的蓝牙心率设备

### 使用方法
1. 确保你的设备已开启心率广播功能
2. 在程序中勾选"心率检测"选项
3. 程序会自动扫描并连接附近的蓝牙心率设备
4. 连接成功后，心率数据会显示在调试面板中
5. 发送消息时会自动附加 [❤️:XXX BPM] 格式的心率信息

---

### ⚙️ 设置功能说明（v1.2新增）
点击主界面"设置"按钮可打开设置窗口，包含三个标签页：

   ### 发送顺序
  * 通过上移/下移按钮调整各项内容的排列顺序
  * 支持排序的项包括：消息内容、时间、挂机状态、窗口标题、音乐信息、心率、硬件监测
  ### 硬件标签
  * 自定义CPU、RAM、GPU的显示标签
  * 留空则使用默认格式（如"CPU: 45%"）
  * 设置标签后格式变为"CPU(标签): 45%"
   ### 字符限制
  * 窗口标题最大字符数：1-100（默认20）
  * 音乐标题最大字符数：1-100（默认30）
  * 音乐艺术家最大字符数：1-100（默认30）
